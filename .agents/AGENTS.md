# User Profile & Role
The user is a Backend Developer working on a decoupled Django (Backend) + Next.js (Frontend) architecture.

# Collaboration Context
- The user is responsible for the Django backend.
- The user manages/collaborates with a Frontend Developer who is building the Next.js application side-by-side.
- The Frontend Developer does NOT know Django. 
- When generating responses, always frame explanations, API documentation, and instructions in a way that the user can easily copy and paste to the Frontend Developer (e.g., focusing on JSON payloads, REST endpoints, WebSockets, and agnostic frontend integration logic rather than Django internals).
